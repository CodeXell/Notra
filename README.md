# **Notra**

Notra is a Windows Note-taking app.
To download, you can view [our website](https://notranotes.netlify.app) or download the app from this repository.\
The website is for downloading the app, where more information may be found.
You can also download our desktop app from this repository.
We recommend downloading the latest installer after every major update. We will regularly release new and useful features as minor updates, updating on these is optional.\

> [!NOTE]
> To update your app: 
> 1. Visit this repository or [our website](https://notranotes.netlify.app) and download the latest installer.\
> 2. Delete the previous Notra App.\
> 3. Go through the install process.\
> 4. After this process, the new Notra App will successfully be installed on your computer.\
> **Your previous notes will remain in the new Notra App as they are saved in your computer's local storage.**
